Boostrap

Bootstrap es un framework de desarrollo web que permite crear sitios y aplicaciones responsivas de forma rápida y sencilla. Proporciona herramientas como diseños predefinidos, componentes visuales y estilos CSS que facilitan la creación de interfaces modernas y adaptables a distintos dispositivos.
